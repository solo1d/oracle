| 运算符 | 描述                  |                                                              |
| :----- | :-------------------- | ------------------------------------------------------------ |
| 1      | `=`                   | 等于                                                         |
| 2      | `!=`,`<>`             | 不等于                                                       |
| 3      | `>`                   | 大于                                                         |
| 4      | `<`                   | 小于                                                         |
| 5      | `>=`                  | 大于等于                                                     |
| 6      | `<=`                  | 小于等于                                                     |
| 7      | IN                    | 等于值列表中的任何值                                         |
| 8      | ANY/SOME/ALL          | 将值与列表或子查询进行比较。它必须以另一个运算符(例如：`=`，`>`，`<`)作为前缀。 |
| 9      | NOT IN                | 不等于值列表中的任何值                                       |
| 10     | [NOT] BETWEEN n AND m | 相当于`[Not] >= n` 且 <`= m`。                               |
| 11     | [NOT] EXISTS          | 如果子查询返回至少一行，则返回`true`                         |
| 12     | `IS [NOT] NULL`       | 测试`NULL`的值                                               |
